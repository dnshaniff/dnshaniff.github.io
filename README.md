# Denis Hanif Prasetya - Portfolio

Personal portfolio website for showcasing my work as a Full-Stack Developer, focused on building internal systems, custom business applications, and practical web solutions for real business operations.

## Overview

This portfolio highlights selected internal system projects built for business workflows across HR, IT, operations, finance, warehouse, legal, audit, maintenance, and other departments.

The website is built as a static frontend, suitable for deployment on GitHub Pages.

## Featured Projects

### MIIMS
Internal management system for PT MATS Internasional Indonesia.

Includes modules for HR, attendance, IT ticketing, and the company’s main business flow from sales and pricing to operations, business development, and finance.

### Cipta Apps
Centralized internal system for PT Cipta Saksama Indonesia.

Supports cross-department workflows including HR, quality, internal audit, MR, engineering, maintenance, warehouse, and operational support.

### GST Apps
Internal system for PT Good Sale Tech.

Supports HR employee data, GA asset management, IT ticketing, legal document control, and structured operational data management.
